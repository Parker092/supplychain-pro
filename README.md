# Documentación Técnica de SupplyChain Pro

## Descripción

Esta carpeta contiene la documentación técnica del proyecto SupplyChain Pro, desarrollado para la materia Diseño de Arquitectura de Sistemas.

La documentación describe los requerimientos, arquitectura, casos de alerta, persistencia, seguridad de red, riesgos, decisiones arquitectónicas, despliegue y pruebas de presentación.

## Archivos principales

- requerimientos.md: define el alcance técnico mínimo del sistema.
- arquitectura.md: describe la estructura y componentes de la arquitectura.
- casos-de-alerta.md: explica los escenarios críticos que debe manejar el sistema.
- pruebas-presentacion.md: contiene la secuencia de pruebas para la defensa final.
- despliegue-github-education.md: describe el uso de GitHub y GitHub Actions.
- despliegue-google-cloud.md: describe una estrategia de despliegue en Google Cloud.
- seguridad-red.md: documenta el aislamiento de servicios y reglas de comunicación.
- persistencia-docker.md: explica el uso de volúmenes Docker para PostgreSQL.
- riesgos-arquitectonicos.md: identifica riesgos técnicos y medidas de mitigación.
- decisiones-arquitectonicas.md: justifica las principales decisiones de diseño.
- glosario.md: define conceptos técnicos relevantes.

## Uso recomendado

Estos documentos deben mantenerse actualizados durante el desarrollo del proyecto. También pueden utilizarse como base para la presentación final y para justificar las decisiones técnicas tomadas durante la implementación.
